# Dr. Agrawal's Dental Health & Implant Centre

A premium, multi-page professional dental clinic website for Dr. Agrawal's Dental Health & Implant Centre, Agra.

## Features
- **Multi-page Architecture**: Separate pages for About, Services, Doctors, Appointment, and Contact.
- **Service Detail Pages**: Dedicated pages for all major dental treatments.
- **Doctor Profiles**: Comprehensive profiles for Dr. Ajay Kumar Agrawal and Dr. Rachana Agrawal.
- **Interactive Appointment System**: Custom-built vanilla JS calendar for booking.
- **WhatsApp Integration**: Seamlessly converts booking requests into professional WhatsApp messages.
- **Premium Design**: Medical-grade aesthetic with smooth animations (GSAP, Lenis, AOS).
- **Fully Responsive**: Optimized for all devices from mobile to ultra-wide desktops.
- **SEO Ready**: Semantic HTML with meta tags and schema markup.

## Tech Stack
- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript
- GSAP (Animations)
- Lenis (Smooth Scroll)
- AOS (Scroll Reveals)

## Structure
```
dental-website/
├── index.html                  # Homepage
├── about.html                  # About the clinic
├── services.html               # All services listing
├── doctors.html                # All doctors listing
├── contact.html                # Contact + Map
├── appointment.html            # Full booking system
├── services/                   # Service detail pages
├── doctors/                    # Doctor profile pages
├── css/                        # Stylesheets
├── js/                         # JavaScript logic
└── images/                     # Image assets (placeholders included)
```

## How to Run
Simply open `index.html` in any modern web browser. No build tools or installation required.
